# House-Rent-Analysis
House Rent Analysis Using Python (EDA Project)

This project explores and analyzes a house rent dataset using Python.  
It includes data cleaning, exploratory data analysis (EDA) of house rent prices.

## Steps
- Data loading and preprocessing
- Missing value and outlier treatment
- Categorical encoding
- Exploratory Data Analysis (EDA)
- Visualization using matplotlib and seaborn
- Correlation and feature insights

## Tools & Libraries
- Python (Pandas, NumPy)
- Matplotlib, Seaborn
- Scikit-learn (if modeling used)

## Key Insights
  - Rent prices are highly influenced by city and size.
  - Semi-furnished houses dominate across most cities.
